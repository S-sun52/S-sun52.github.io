我的hexo博客优化

2020-02-19 | ℃
|

在windows下,关于我的hexo博客的搭建

  • 现在网上有很多关于hexo博客搭建的视频和教程，我的搭建是参照： bi站up主CodeSheep的搭建视频。
    http://www.bilibili.com/video/av44544186/

    在windows下，matery主题优化

  • 下面是我进行的几项简单的blog优化，因为我认为blog主要看重的是内容。
  • 注意，您更改hexo下的设置后，在cmd上执行：
    hexo clean后hexo g后hexo s后hexo d，就可推到您的静态blog，因为操作一样，后面几不做说明。

    我的博客预览

  • 我的静态博客地址：
    https://s-sun52.github.io/

    主题更换

  • 在浏览其他人的博客时，发现有一个主题被人们广为引用——Matery。
  • 更改方法：
    1. 下载： 将主题matery下载，在windows的cmd下的命令：git clone https://github.com/blinkfox/hexo-theme-matery,然后将这个文件放到你的blog文件的themes文件夹下，可以直接将名字改为matery,以便后面简便使用。
    2. 再到hexo下的配置文件_config.yml中（可用记事本打开），将theme:的内容，更改为matery,注意，在冒号后有一空格。
    3. 然后，在cmd上执行：
      hexo clean后hexo g后hexo s后hexo d.

      更改主题内容，使之个性化

      更改logo和博客标题

  • 对于博客标题的修改： 在hexo下的配置文件_config.yml中，直接修改title:后的内容即可。
  • 对于头像的修改： 发现matery已经给我们配置好了一个头像，在主题文件matery下面的_config.yml中，发现他将logo配置为在matery\source\medias下的logo.png图片，为了方便，就可以直接找一个你喜欢的logo来将原来的logo.png替换。

    将主题设置为中文

  • 将hexo下的配置文件_config.yml中，language: 后的内容修改为空格+zh-CN，就可。

    修改主题背景

  • 在matery主题的配置文件_config.yml中，已经设置了背景为matery\source\medias下的图片0.jpg,1.jpg,2.jpg；所以为了方便，我们就可以直接将自己想要的图片修改为与这几个图片相似的像素和存储大小，直接将这几张图片替换。

    修改首页显示文字

  • 在matery下的_config.yml文件中，修改dream下的title:和text:后面的文字即可。

修改首页subtitle下的文字

  • 在matery下的_config.yml文件中，修改sub1:和sub2:后的内容。

    开启背景动态彩带

  • 在matery下的_config.yml文件中，修改ribbon_dynamic:下将enable:后的内容改为true即可。

    开启评论

  • 在matery下的_config.yml文件中，将各个评论的配置激活便可。

    添加卡通人物

  • 卡通人物原作者： https://huaji8.top/post/live2d-plugin-2.0/，在这里面可以挑选自己喜欢的卡通人物。（记住它的名字哦）

  • 下载文件： 在cmd中，输入npm install –save hexo-helper-live2d 来下载。
